# Contexto do Projeto: Landing Page Contabiliza+


## Stack Tecnológica (Obrigatória)

- **Frontend:** HTML5, JavaScript

- **Estilização:** Sass (SCSS) - Compilando para CSS.

- **Backend:** PHP para processamento de formulários.

- **Banco de Dados:** MySQL.


## Regras de Design e Comportamento

- **Estilo:** Moderno, contabilidade digital, paleta de cores verde e dark (conforme imagens).

- **Animações:** Devem ser feitas via CSS (transições/keyframes) gatilhadas por scroll ou hover.

- **Responsividade:** Mobile-first é essencial.

- **Interação:** O carrossel deve permitir hover nos cards sem que o conteúdo seja cortado pelo container pai.


## Estrutura de Arquivos Preferencial

- `/scss`: Arquivos .scss (separar variáveis, mixins e componentes).

- `/js`: Scripts de animação e carrossel.

- `/php`: Lógica de conexão e persistência.

- `index.php`: Arquivo principal.
